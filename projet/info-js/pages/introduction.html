Introduction

JavaScript is a text-based programming language used both on the client-side and server-side that allows you to make web pages interactive. Where HTML and CSS are languages that give structure and style to web pages, JavaScript gives web pages interactive elements that engage a user.

Link javascript to html :


On page script


<!DOCTYPE html >
<title>HTML Tutorial </title>
<body>
                  
<p>This is a paragraph.</p>
                  
<script type="text/JavaScript"> alert("hello word") </script>
</body>
</html>

Example Explained:

  • The script tag is what we use to includes our JavaScript. It's a lot like the link tag you've already been using to include your CSS files.
  • Type Attribute is used to specify the type of script and identify the content of the Tag. It has a Default value which is “text/javascript”.
  • The Alert() method displays an alert box with a message and an OK button.

Include external JS file


<!DOCTYPE html >
<title>HTML Tutorial </title>
<body>
                  
<p>This is a paragraph.</p>
                  
<script src="filename.js"> </script>
</body>
</html>

Example Explained:

  • SRC attribute specifies the URL of an external script file.
