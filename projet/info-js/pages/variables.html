JavaScript Variables

4 Ways to Declare a JavaScript Variable:

  • Using var .
  • Using let .
  • Using const .
  • Using no declaration key .


Declare with var

Example

var x = 10 ;
var y = 11 ;
var z = x + y;

When to Use JavaScript VAR?

Always declare JavaScript variables with var, let, or const. The var keyword is used in all JavaScript code from 1995 to 2015.
The let and const keywords were added to JavaScript in 2015. If you want your code to run in older browser, you must use var.

Declare with let

Example

let x = 10 ;
let y = 11 ;
let z = x + y;

VAR or LET ?

The let keyword is used to declare variables in JavaScript. The var keyword can also be used to declare variables, but the key difference between them lies in their scopes.
var is function scoped while let is block scoped.

Declare with const

Example

const pi = 3.14 ;
const taux = 12 ;

When to Use JavaScript CONST ?

The const declaration create a named constant that is read-only. This does not mean that the contained value is immutable, just that the identifier cannot be reassigned.
In other words, the value of a constant cannot be changed by subsequent reassignments. A constant cannot be redeclared.


Be careful !

You cannot re-declare a variable declared with CONST or LET.
This will not work:
Example

let x = 12 ;
let x ;
