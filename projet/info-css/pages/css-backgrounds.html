Backgrounds

The CSS background properties are used to add background effects for elements.

In this chapter,we have mani properties but we will focus only on two background properties:

  • background-color
  • background-image
Background color

The background-color property specifies the background color of an element.

Example

h1 {
background-color: green;
}
p {
background-color: yellow;
}

Background Image Repeat

By default, the background-image property repeats an image both horizontally and vertically.

Tip: To repeat an image vertically, set background-repeat: repeat-y;

Tip: To repeat an image horizentally, set background-repeat: repeat-x;

Tip: Showing the background image only once is also specified by the background-repeat property:

Example

body {
background-image: url("img_tree.png");
background-repeat: no-repeat;
}
