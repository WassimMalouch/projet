Margin & Padding

Margin

CSS has properties for specifying the margin for each side of an element:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

CSS Demo: Syntax

Margin - Shorthand Property

The margin property is a shorthand property for the individual margin properties.

So, here is how it works:If the margin property has four values:

  • margin: 25px 50px 75px 100px;
    • top margin is 25px
    • right margin is 50px
    • bottom margin is 75px
    • left margin is 100px
Example

p {
margin: 25px 50px 75px 100px;
}

If the margin property has four values:

  • margin: 25px 50px 75px;
    • top margin is 25px
    • right margin is 50px
    • bottom margin is 75px
Example

p {
margin: 25px 50px 75px;
}
Padding

The CSS padding properties are used to generate space around an element's content, inside of any defined borders.

CSS has properties for specifying the padding for each side of an element:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

CSS Demo: Padding

Padding - Shorthand Property

The padding property is a shorthand property for the individual padding properties.

So, here is how it works:If the margin property has four values:

  • padding: 25px 50px 75px 100px;
    • top padding is 25px
    • right padding is 50px
    • bottom padding is 75px
    • left padding is 100px
Example

p {
padding: 25px 50px 75px 100px;
}

If the padding property has four values:

  • padding: 25px 50px 75px;
    • top padding is 25px
    • right and left padding is 50px
    • bottom padding is 75px
Example

p {
padding: 25px 50px 75px;
}
